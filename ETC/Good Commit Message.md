# 좋은 git 커밋 메시지

1. 제목과 본문을 한 줄 띄워 분리하기
2. 제목은 영문 기준 50자 이내로
3. 제목 첫글자를 대문자로
4. 제목 끝에 . 금지
5. 제목은 명령조로
6. 본문은 영문 기준 72자마다 줄 바꾸기
7. 본문은 어떻게보다 무엇을, 왜에 맞춰 작성하기


- 커밋 메시지로 Github 이슈(issue)를 자동 종료시키기
- Github가 이슈 종료로 인식하는 키워드는 다음과 같습니다.
	- close
	- closes
	- closed
	- fix
	- fixes
	- fixed
	- resolve
	- resolves
	- resolved
